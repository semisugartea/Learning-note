# 識別字 identifier

	用來識別特殊物件,例如類別(class),函數(function),變數(variable)等

1.  由字母（A-Z,a-z）、數字（0-9）、**單一**個下劃線「_」組成，並且首字符不能是數字，但可以是字母或者下劃線。

2.	第一個字元不可為數字

3.	不能把關鍵字作為識別字

4.	在識別字中不可有特殊符號例如!, @, #, $, %

5.	識別字長度不限

# 縮排 indentation

	縮排的空格數是可變的,但同一代碼區的空格數必須相同,否則可能導致IndentationError
